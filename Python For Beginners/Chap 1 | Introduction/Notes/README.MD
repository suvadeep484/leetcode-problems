# Introduction

## Hello World

- To print `Hello, world!` we use the following syntax 

  ```py
  print("Hello, world!")
  ```

  

## What is Python ? 

- Python is an **interpreted** programming language

## Interpreted Programming Language vs Compiled Language 

- Compiled Language
  - In a compiled language, the code is translated into machine code before it is run
- Interpreted Language 
  -  In an interpreted language, the code is executed line by line by an interpreter, usually written in another language

## How can we print `They said, "Hello, world!"`

- We can use a backspace character 

  ```python
  print("They said, \"Hello, world!\"")
  ```

- backslash `\` tells Python that we want to interpret the quote `"` as a charcater inside the string,  not as a quote that ends the string 

## Errors 

- Syntax Error 
  - Syntax errors occur when the code is not written correctly according to the rules of the programming language
- Runtime Error 
  - A runtime error is an error that occurs while the program is running. That means the syntax of the program itself is fine. It is usually caused by something that the programmer did not anticipate
  - For example, if we programmed a calculator and the user tries to divide by zero, we will get a runtime error
- Logical Error 
  - A logical error is an error that occurs when the program runs without crashing or producing an error, but the output is not what the programmer intended.

## Comments 

- In Python, comments are created using the `#` character. Any text after the `#` character on a line is considered a comment.

  ```py
  # This is a comment
  ```
